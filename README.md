# <p align="center">Packeteer</p>

<p align="center">A simple, yet powerful type-safe networking library for Roblox Luau.  </p>

![License](https://img.shields.io/github/license/NotDumbDev/Packeteer?style=for-the-badge)


## ✨ Features

- 🧩 **Schema-driven.**  
  Packeteer uses a networking schema as the single source of truth for defining communication between the server and client.
- 🛡️ **Type-safe.**  
  Define packet data using Luau types and get autocomplete and type checking throughout your networking code.
- ⚡ **Promise-based.**  
  Packet Functions use promises, making asynchronous remote communication cleaner and easier to handle.
- 🪶 **Minimal boilerplate.**  
  No manually creating RemoteEvents, RemoteFunctions, or managing networking instances. Define your packets and let Packeteer handle the rest.
